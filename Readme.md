# Projet Optimisation Combinatoire
## Sac à dos + Algorithme génétique + Machine Learning

Ce projet propose une approche hybride pour le problème du sac à dos.

## Objectif
Comparer :
- un algorithme génétique classique
- un algorithme génétique guidé par un modèle de machine learning

## Structure
```text
projet-sac-a-dos-ml/
│
├── data/
├── results/

└── main.py

## Instructions d’exécution

Créer et activer l’environnement virtuel :

### Windows
```bash
python -m venv venv
venv\Scripts\activate
```

Installer les dépendances :

```bash
python -m pip install -r requirements.txt
```

Lancer le projet :

```bash
python main.py
```
